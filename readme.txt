1.VGGish文件夹装的是用VGGish网络提取音频特征的代码，为TensorFlow+Keras版本，具体使用方法阅读该文件夹下的readme.txt和README.MD。

2.pseudo-3d-pytorch文件夹装的是pytorch版本的P3D网络代码，具体使用方法阅读该文件夹下的README.md。

3.fusion网络是特征融合的代码，其中，DMRN_fusion是使用基于DMRN的特征层融合方法的代码，FC_fusion是使用基于直接级联的特征层融合代码。决策层融合的在classification文件夹中。

4.classification文件夹是分类器模型的代码。其中，

classification_violence_DL.py是使用基于决策层融合方法的暴力场景检测分类器。
classification_violence_FC.py是使用基于直接级联的特征层融合方法的暴力场景检测分类器。
classification_violence_DMRN.py是使用基于DMRN的特征层融合方法的暴力场景检测分类器。

classification_valence_DL.py是使用基于决策层融合方法的情感效价等级分类器。
classification_valence_FC.py是使用基于直接级联的特征层融合方法的情感效价等级分类器。
classification_valence_DMRN.py是使用基于DMRN的特征层融合方法的情感效价等级分类器。

classification_arousal_DL.py是使用基于决策层融合方法的情感唤醒度等级分类器。
classification_arousal_FC.py是使用基于直接级联的特征层融合方法的情感唤醒度等级分类器。
classification_arousal_DMRN.py是使用基于DMRN的特征层融合方法的情感唤醒度等级分类器。

multitask1_FC.py是基于直接级联的特征层融合方法的暴力场景检测+情感效价等级的多任务训练分类器。
multitask1_DMRN是基于DMRN.py的特征层融合方法的暴力场景检测+情感效价等级的多任务训练分类器。

multitask2_FC.py是基于直接级联的特征层融合方法的暴力场景检测+情感唤醒度等级的多任务训练分类器。
multitask2_DMRN.py是基于DMRN的特征层融合方法的暴力场景检测+情感唤醒度等级的多任务训练分类器。

multitask3_FC.py是基于直接级联的特征层融合方法的情感效价等级+情感唤醒度等级的多任务训练分类器。
multitask3_DMRN.py是基于DMRN的特征层融合方法的情感效价等级+情感唤醒度等级的多任务训练分类器。

multitask4_FC.py是基于直接级联的特征层融合方法的暴力场景检测+情感效价等级+情感唤醒度等级的多任务训练分类器。
multitask4_DMRN.py是基于DMRN的特征层融合方法的暴力场景检测+情感效价等级+情感唤醒度等级的多任务训练分类器。

makedataset.py是制作训练集和测试集的代码。

testcode_vio.py是对暴力场景检测任务进行测试的代码。
testcode_val.py是对情绪效价等级任务进行测试的代码。
testcode_aro.py是对情绪唤醒度等级任务进行测试的代码。

trec_eval.exe是用于计算mAP值的外部程序，已在python程序内直接调用。
cygwin1.dll是配合trec_eval.exe使用的外部拓展程序。
